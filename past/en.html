How to Use

This is a Renju tournament statistics tool, created as a hobby project by @nachirenju. It may contain bugs, and I cannot take responsibility for any issues caused by its use. Please contact me on X if needed. I also run a YouTube channel unrelated to this tool — your support is appreciated!
I translated this tool with ChatGPT. Some translations might not be correct. Feel free to contact me if you have any questions.

Basic Usage

  1. Click “Choose File” to load a database (.rif or .xml). (Databases can be downloaded from RIF official.)
  2. Search tournaments by keyword or input Tournament IDs directly (comma separated).
  3. Click “Analyze Tournament” to display basic statistics.
  4. Currently supported rules: Soosrv-8, Taraguchi-10, Classic, Yamaguchi.
  5. Player filter: restrict tournaments by players (supports multiple players).

About the Data

  • Move-by-move win counts exclude draws (only decisive games are counted).
  • In the frequent sequence list, notation like Black(1,3) means the black player played moves 1 and 3 (swap history).

Player Statistics

  • You can search by name, surname, or native name from the RIF database.
  • Supports multiple players (comma separated).
  • When multiple players are entered, only games between them are extracted. (Example: input "A,B" → only A vs B games. Input "A,B,C" → AB, AC, BC combinations.)

Swap History Search

  • Shows the swap history and who played which move.
  • Currently, Taraguchi-10 10-move display is not supported.
